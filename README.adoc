== PURPOSE:
The purpose of this reference architecture is to demonstrate capabilities of BPM Suite 6 _Service Tasks_ to invoke remote SOA services.
The transports utilized to invoke these remote SOA services are:  REST, SOAP and JMS.
Advanced use cases of each of these transports are investigated.
